# INDAY
An Arduino-based wearable device for the blind 

# Hardware Requirment
- Arduino Uno
- MAX9812 Microphone Voice Module
- ESP8266 Wifi Module
- NEO-6M GPS Module
- HC-SR04 Ultrasonic Distance Sensor
- Emic 2 Text-to-Speech Module

# Software Requirment
- Arduino IDE

# Features
- Listen to Voice Commands
- Detect Obstacles
- Getting Current Location
- Voice Output
